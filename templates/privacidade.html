Politica de Privacidade

Ultima atualizacao: 28/02/2026

1. Dados coletados

Coletamos dados fornecidos pelos usuarios e clientes para gerar contratos e operar a plataforma.

2. Finalidade

Os dados sao utilizados para criacao de contratos, comunicacoes operacionais e melhoria do servico.

3. Compartilhamento

Podemos compartilhar dados com provedores de infraestrutura estritamente necessarios para o servico.

4. Retencao

Os dados sao armazenados pelo tempo necessario para cumprir as finalidades do servico.

5. Direitos do titular

O titular pode solicitar acesso, correcao ou exclusao de seus dados.

6. Seguranca

Adotamos medidas tecnicas e organizacionais para proteger os dados.

7. Contato

Para assuntos de privacidade, contate: privacidade@preencheai.com
